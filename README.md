# Trader Performance Analysis using Market Sentiment

## Objective
This project analyzes the relationship between trader performance and market sentiment using historical trading data and the Fear & Greed Index.

## Datasets
- Historical trader data
- Fear & Greed sentiment data

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code

## Analysis Performed
- Data cleaning and preprocessing
- Date conversion and dataset merging
- Average PnL by sentiment
- Total PnL by sentiment
- Win rate by sentiment
- Profit vs Fear & Greed Index value
- Average profit by side
- Direction-based performance analysis

## Key Insights
- Fear generated the highest total profit.
- Extreme Greed showed the highest average profit per trade.
- Extreme Greed also had the highest win rate.
- SELL trades showed higher average profit than BUY trades.
- Trader profitability changes with both sentiment and trade direction.

## Conclusion
Market sentiment has a noticeable impact on trader performance. Strong sentiment conditions such as Fear and Extreme Greed created better trading outcomes, suggesting that sentiment indicators can improve trading decision-making.
